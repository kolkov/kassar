Сервисные услуги по обслуживанию онлайн касс

  • Подключение эквайринга.
  • Программирование номенклатуры контрольно-кассовой техники (ККТ).
  • Обучение работе на ККТ.
  • Подключение к оператору фискальных данных (ОФД).
  • Электронная подпись.
  • Регистрация в Инспекции Федеральной налоговой службы (ИФНС).
  • Выезд сервисного инженера (1 час работы).
  • Удаленная работа сервисного инженера (1 час работы).
  • Снятие ККТ с учета в налоговой инспекции.
  • Перерегистрация ККТ.
  • Информационно-техническое сопровождение ККТ с выездом специалиста (3 месяца).
  • Информационно-техническое сопровождение ККТ с выездом специалиста (12 месяцев).
  • Удаленное техническое сопровождение (12 месяцев).
